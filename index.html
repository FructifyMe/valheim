My Little UI

Client Side Server Synced

A collection of quality-of-life interface tweaks designed to give you more information without cluttering your screen. Includes comprehensive tooltips, weather forecasts, and inventory management tools.

Note: Most config values are synchronized with the server. You can tweak visual preferences using Configuration Manager (F1).

HUD Enhancements

  • In-game Clock: Moveable, shows time/day
  • Weather Forecast: Tracks rain, mist, storms
  • Wind Forecast: Shows next 5 wind directions
  • Weight & Slots: Bar above food slots

Crafting & Items

  • Multicraft support (Gamepad & Mouse)
  • Crafting list filters & sorting
  • Resource requirement counts in inventory
  • Colored durability bars
  • Quick repair: Hold 'Use' to repair all

Smart Tooltips

  • Fermenter: Time remaining, percent done
  • Plants: Time to full growth
  • Smelters/Kilns: Queue info & time to finish
  • Tames: Growth time, hunger status
  • Combined Armor & Status Effect stats

Storage Management

  • Custom Chest Names
  • Show chest contents on hover
  • Toggle free slots vs item count
  • Fast filtering in chests

Serverside QoL

Server Only

Powerful backend tweaks that run entirely on the server.

Xbox & Vanilla Compatible
Crossplay Friendly Designed for dedicated servers with mixed PC/Xbox players.
Performance Fixes ownership issues for smelters/cooking on bad connections.
Admin Room Includes an in-game room for admins to toggle features via signs.

World & Travel

  • Auto-generated Portal Hub
  • Map Pins for Portals and Ships (Auto-updating)
  • Unlock Traders (removes progression lock)
  • Mist removal in Mistlands (Configurable)

Base Automation

  • Auto-sort inventories into nearby containers
  • Auto-pickup dropped items into chests
  • Smelters/Kilns pull fuel/ore from nearby chests
  • Crafting Stations: Extended range & extensions
  • Doors auto-close after use

Tames & Creatures

  • Commandable Tames (Like wolves)
  • Teleport following tames with you
  • Tames are always fed (Never hungry)
  • Show taming/growing progress to nearby players

Quality of Life

  • Infinite Fuel for fireplaces/torches
  • Torches/Fires ignore rain
  • Infinite Stamina (Building/Farming modes)
  • Assignable ownership for interactables
  • In-world config room for Admins

* Features marked here are available but depend on the server admin's specific configuration.
